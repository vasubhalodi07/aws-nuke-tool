Steps to Get AWS Keys and Account ID

Step 1: Sign in to the AWS Management Console

Open the AWS Management Console and sign in with your AWS credentials.

Step 2: Navigate to IAM (Identity and Access Management)

In the AWS Management Console, search for and select IAM from the services menu.

Step 3: Create a New User or Select an Existing User

To create a new user:
1. Select Users from the sidebar.
2. Click Add user.
3. Enter a username and select the Programmatic access checkbox.
4. To use an existing user, select the user from the list.

Step 4: Attach Policies

Ensure the user has the necessary permissions by attaching policies. For aws-nuke, attach policies like AdministratorAccess or create a custom policy with the required permissions.

Step 5: Retrieve Access Keys

After creating the user or selecting an existing user, go to the Security credentials tab and click Create access key or Manage access keys.

Copy the Access key ID and Secret access key. Store them securely.

Note: Treat your access keys like a password. Do not share them or expose them in public repositories.

Step 6: Find Your AWS Account ID

To find your AWS account ID:
1. Click on your account name or number at the top right corner of the AWS Management Console.
2. Select My Account.
3. Your AWS account ID is listed at the top under Account settings.
